## App Description

[Backend Framework: Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash

# backend development
$ npm run start

# watch mode
$ npm run start:dev
```

Open [http://localhost:4000](http://localhost:4000) with your browser to see the result.

## Run tests

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
